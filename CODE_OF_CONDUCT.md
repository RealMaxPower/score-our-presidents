# Code of Conduct

This project adopts the **Contributor Covenant, version 2.1**, available at:

https://www.contributor-covenant.org/version/2/1/code_of_conduct/

The full canonical text governs participation in this repository, in issues and pull requests, and in any project-related communications (including the editorial address).

## Reporting

To report behavior that violates this Code of Conduct, write to:

**conduct@scoreourpresidents.org**

Reports are handled confidentially. The maintainer will acknowledge a report within seven days and communicate any action taken (or a reasoned decision not to act) within thirty days. Reporters are not retaliated against for good-faith reports.

## Scope

This Code applies to all project spaces — the repository, its issues and pull requests, the editorial inbox, and any public communication made on behalf of the project. It also applies when an individual is officially representing the project in public.

## Enforcement

The maintainer is responsible for enforcement and may, depending on severity: ask a contributor to edit or remove content; close issues or pull requests; warn; suspend interaction; or ban. Enforcement decisions follow the Community Impact Guidelines in the Contributor Covenant 2.1 text linked above.
